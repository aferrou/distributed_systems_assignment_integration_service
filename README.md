<h1>FitTrack Integration Service</h1>
<h2>Overview</h2>

This service provides real-time weather forecasts for the [FitTrack fitness platform](https://github.com/aferrou/distributed_systems_assignment_2025_26), specifically for outdoor training sessions.

<h3>Setup & Run</h3>

---

**<h4>Run Locally</h4>**

1. Clone the repository
```
git clone https://github.com/aferrou/distributed_systems_assignment_integration_service.git
```

2. Build the project
```
mvn clean install
```

3. Start the application
```
mvn spring-boot:run
```
---
Open in browser
```
http://localhost:8081/swagger-ui/index.html
```
